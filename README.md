### Run Locally

To run **pedacinho-do-ceu** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/estevam5s/pedacinho-do-ceu.git
```

Windows:

```bash
git clone https://github.com/estevam5s/pedacinho-do-ceu.git
```
